Personal Rule

Full-Time Trader Standard

  1. Is H1/M15 structure clear?
  2. Is this a real setup or only M5 noise?
  3. Am I reversing emotionally?
  4. Has 30 minutes passed if this is opposite direction?
  5. Is this within my daily trade limit?
  6. Can I accept the SL calmly?
  7. Will I still respect myself if this trade loses?

If the answer is not clean, stay flat. Being flat is also a professional position.
