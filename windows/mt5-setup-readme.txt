FX Edge Journal MT5 Connector

This installer registers a local Windows URL protocol:

  fxedge-mt5://open

After setup, the MT5 icon inside FX Edge Journal can open your local MetaTrader 5 terminal64.exe program from supported desktop browsers.

If MetaTrader 5 is installed in a custom folder, rerun this installer and browse to terminal64.exe when asked.

The manual fallback remains available:

  install-fxedge-mt5-protocol.bat
